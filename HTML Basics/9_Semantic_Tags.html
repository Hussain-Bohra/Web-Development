What are Semantic Elements?

A semantic elements clearly describes its meaning to both the browser and the developer.

Semantic elements = elements with a meaning.

Example of non-semantic elements: <div> and <span> - Tells nothing about its content.

Example of semantic elements: <form> , <table> and <article> - Clearly definees its content.

Semantic Elements in HTML

Many web sites contain HTML code like: <div id="nav"> <div class="header"> <div id="footer"> to indicate navigation, header, and footer.

In HTML there are some semantic elements that can be used to define different parts of a web page:

  • <article>
  • <aside>
  • <details>
  • <figcaption>
  • <figure>
  • <footer>
  • <header>
  • <main>
  • <mark>
  • <nav>
  • <section>
  • <summary>
  • <time>
